# 1st Samwster-ICT-project# 📱 iPhone Comparison Hub

A simple and informative website developed as an ICT course project to help users compare different iPhone models. The website presents side-by-side comparisons of specifications, features, and improvements across multiple iPhone generations.

## 🌟 Features

- Compare iPhone models easily
- Clean and user-friendly interface
- Responsive page layout using HTML and CSS
- Dedicated comparison pages for different iPhone generations
- Reviews section
- About Us section
- Hosted using GitHub Pages

## 📋 Comparisons Included

- iPhone 11 vs iPhone 12
- iPhone 12 vs iPhone 13
- iPhone 13 vs iPhone 14
- iPhone 14 vs iPhone 15
- iPhone 15 vs iPhone 16

## 🛠️ Technologies Used

- HTML5
- CSS3
- Git
- GitHub
- GitHub Pages

## 📖 Project Objective

The objective of this project is to provide users with a simple platform for comparing different iPhone models based on specifications such as:

- Display
- Processor
- Camera
- Design
- Performance
- Features

This helps users understand the differences between iPhone generations and make informed decisions.
